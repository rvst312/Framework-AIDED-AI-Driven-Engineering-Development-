# Preguntas Frecuentes

Respuestas a las dudas más comunes sobre el uso del Framework AIDED.

## General

### ¿Es AIDED compatible con otros frameworks además de Next.js?
Aunque AIDED está optimizado para Next.js debido a sus capacidades de Server Components, los principios fundamentales pueden aplicarse a otros frameworks como Remix o Nuxt.

### ¿Necesito una suscripción a herramientas de IA?
Sí, para aprovechar al máximo el framework recomendamos acceso a modelos avanzados como GPT-4 o Claude 3 Opus, ya sea a través de ChatGPT Plus, Claude Pro o APIs directas.

## Técnico

### ¿Puedo usar otra base de datos en lugar de Supabase?
El framework está diseñado con Supabase en mente por su facilidad de uso y capacidades de tiempo real, pero puedes adaptar la capa de datos para usar cualquier base de datos PostgreSQL o incluso otras soluciones SQL/NoSQL.

### ¿Cómo manejo las migraciones de base de datos?
Recomendamos usar el CLI de Supabase para gestionar migraciones locales y desplegarlas a producción a través de tu pipeline de CI/CD.

## Metodología

### ¿Cuánto tiempo toma completar la Fase 0?
Depende de la complejidad del proyecto, pero típicamente toma entre 1 y 3 días tener una conceptualización y arquitectura completa y validada.
