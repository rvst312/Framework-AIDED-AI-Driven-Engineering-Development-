# Changelog

Mantente al día con las últimas actualizaciones del Framework AIDED.

## v1.2.3 (Actual)
- **Nuevo**: Integración nativa con modelos de IA de última generación.
- **Mejora**: Optimización del pipeline de CI/CD para despliegues más rápidos.
- **Fix**: Resolución de problemas de caché en componentes de servidor.

## v1.2.0
- **Nuevo**: Introducción del sistema de Agentes Especializados.
- **Nuevo**: Soporte para bases de datos vectoriales en Supabase.
- **Mejora**: Documentación ampliada para la Fase 0.

## v1.1.0
- **Nuevo**: Plantillas de inicio rápido para Next.js 14.
- **Mejora**: Refactorización de componentes UI base.
- **Fix**: Correcciones menores en la autenticación.

## v1.0.0
- **Lanzamiento Inicial**: Publicación del Framework AIDED.
- Metodología completa de 6 fases.
- Guías de integración con IA.
