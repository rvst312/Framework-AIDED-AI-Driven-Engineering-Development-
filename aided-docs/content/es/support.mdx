# Soporte

Estamos aquí para ayudarte a tener éxito con el Framework AIDED.

## Canales de Soporte

### Documentación
Tu primera parada debe ser siempre esta documentación. Hemos cubierto la mayoría de los casos de uso y problemas comunes. Usa la barra de búsqueda (⌘F) para encontrar lo que necesitas rápidamente.

### Comunidad
Únete a nuestro servidor de Discord para conectar con otros desarrolladores, compartir tus proyectos y obtener ayuda de la comunidad.

### Soporte Premium
Para equipos Enterprise, ofrecemos planes de soporte dedicados con SLAs garantizados y acceso directo a nuestros ingenieros de soluciones.

## Reportar Problemas

Si encuentras un bug en el framework o en la documentación, por favor abre un issue en nuestro repositorio de GitHub. Incluye tanta información como sea posible para ayudarnos a reproducir y solucionar el problema.
